### 배경

- 회사에서 인터넷이 안되는 내부망 환경에서 docker 설치가 필요함
- 사내 인터넷 VDI로 virtual box를 설치하는 건 어려울 것 같음
- 집에서 virtual box를 설치하고 동일한 OS를 세팅한 뒤 도커를 설치, 해당 패키들을 별도 파일로 저장하여 회사에 반입하고자 함
